Throws a std::runtime_error in debug builds if the condition is false. In release builds, logs a warning message instead.

This macro checks the expression expr.

  • If expr evaluates to false in debug mode (NDEBUG not defined), it throws an exception with the provided message msg.
  • If in release mode (NDEBUG defined), it prints a warning to std::cerr but does not throw.
Parameters
exprThe condition to evaluate.
msgThe error message used for the exception or warning.
Note
Use this macro to enforce debug-only sanity checks without crashing release builds.
