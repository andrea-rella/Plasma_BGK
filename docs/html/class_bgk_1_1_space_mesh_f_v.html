Detailed Description

template<typename T>
class Bgk::SpaceMeshFV< T >

1D Finite Volume Spache Mesh Class (extends BaseMesh<T, std::vector<T>>)

This class implements a 1D finite volume space mesh for the BGK model. It defines the computational points and volume boundaries based on the configuration data provided.

◆ initialize_mesh()

template<typename T>
void Bgk::SpaceMeshFV< T >::initialize_mesh ( )
overridevirtual

Initializes the mesh the mesh by filling x_comp and x_vol vectors with the default spacing (see note).

This method computes the computational points and volume boundaries based on the configuration data calling initialize_with_custom_spacing. It uses the polynomial spacing for the first N0 points and uniform spacing for the remaining points.

Exceptions
std::runtime_errorif the SpaceMeshFV object is not constructed.
Note
The mesh is initialized with the following formula [aoki1990numerical]

\begin{align*} x_i = d_1 i + 0.25 (d_2-d_1)(i^4/N_0^3), \quad i = 0,\dots,N_0 \\ x_i = x_{N_0} + (i-N_0) d_2, \quad i = N_0+1,\dots,N \\ x_{i+1/2} = 0.5 (x_i + x_{i+1}), \quad i = 0,\dots,N-1 \end{align*}

Implements Bgk::BaseMesh1D< T, std::vector< T >, MeshNature::SPACE >.

◆ QUICK_coefficients_n()

template<typename T>
std::vector< std::pair< T, T > > QUICK_coefficients_n ( const SpaceMeshFV< T > & mesh)
related

Computes QUICK coefficients for a given SpaceMeshFV and a NEGATIVE velocity.

Computes the QUICK (quadratic upwind interpolation) coefficients \( a_1, a_2 \) for a given SpaceMeshFV mesh such that \( \phi_e = \phi_U + a_1 (\phi_D-\phi_U) + a_2 (\phi_U-\phi_{UU}) \), where:

  • \( a_1 = \frac{ (x_e-x_U) (x_e-x_{UU}) }{ (x_D-x_U) (x_D-x_{UU}) } \)
  • \( a_2 = \frac{ (x_e-x_U) (x_D-x_e) }{ (x_U-x_{UU}) (x_D-x_{UU}) } \)

with U = Upstream, D = Downstream, UU = Upstream-Upstream, e = Edge.

Template Parameters
Tprecision type of the mesh components.
Parameters
meshSpace mesh to compute the coefficients for.
Returns
std::vector<std::pair<T, T>>
Exceptions
std::runtime_errorif the mesh is not initialized or constructed.
Note
The first and last elements of the vector are set to (0., 0.) as they are not used in this specific finite volume formulation of the problem.
